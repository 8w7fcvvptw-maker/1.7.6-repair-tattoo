# INK STUDIO — Сайт записи на тату-услуги

Одностраничный лендинг тату-студии с формой записи на сеанс. Чистый HTML + CSS + JavaScript, без фреймворков и зависимостей.

## Возможности

- Адаптивный дизайн (mobile-first) с тёмной темой
- Секции: услуги и цены, портфолио с lightbox-галереей, о мастере, форма записи
- Форма записи с валидацией и отправкой заявки в Telegram
- Плавные анимации при скролле (Intersection Observer)
- Маска ввода для номера телефона
- Мобильное гамбургер-меню

## Быстрый старт

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере (или запустите через Live Server в VS Code)

Сайт полностью статический — сервер не нужен.

## Настройка отправки в Telegram

Чтобы заявки с формы приходили вам в Telegram:

### 1. Создайте бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный **API Token** (формат: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)

### 2. Узнайте свой Chat ID

1. Напишите любое сообщение своему боту
2. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Найдите в ответе `"chat":{"id": 123456789}` — это ваш **Chat ID**

### 3. Вставьте данные в код

Откройте файл `js/main.js` и замените значения в начале файла:

```js
const TELEGRAM_BOT_TOKEN = 'ваш_токен_бота';
const TELEGRAM_CHAT_ID = 'ваш_chat_id';
```

После этого все заявки с формы будут приходить вам в Telegram.

## Структура проекта

```
├── index.html      — главная страница
├── css/
│   └── style.css   — все стили
├── js/
│   └── main.js     — логика (навигация, галерея, форма, анимации)
├── images/         — SVG-заглушки портфолио (замените на свои фото)
└── README.md
```

## Замена изображений

Замените SVG-файлы в папке `images/` на реальные фотографии работ (JPG/PNG/WebP). Имена файлов в `index.html`:

- `images/work-1.svg` ... `images/work-6.svg` — работы портфолио
- `images/master.svg` — фото мастера
